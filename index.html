Conhecimentos

HTML

É código de marcação de texto, que é utilizado na criação de páginas web. Tenho um ano e seis meses de expêriencia com essa tecnologia.

CSS

É uma liguagem de marcação, utilizada na formatação de textos, no layout das páginas web, e no desenvonvimento das cores do site. CSS, cuida do estilo das páginas web. Eu tenho um ano e seis meses de exepêriencia com essa tecnologia.

JavaScript

É uma liguagem de programação utilizada no desenvonvimento de páginas web. JavaScript é mas aplicada na parte visual dos sites, essa linguagem é responsável pelas animações das páginas web. Eu tenho um ano de exepêriencia com essa tecnologia.

TypeScript

Mais conhecido como um superset do Javascript, ou seja, um conjunto de ferramentas, o TypeScript foi criado com o objetivo de incluir recursos que não estão presentes no JS. Por meio dele é possível definir a tipagem estática, parâmetros e retorno de funções., eu tenho seis meses de exepêriencia com essa tecnologia. E continuo me aprimorando, e estudando essa ferramenta poderosa.

Angular

O AngularJS é um framework de código aberto usado para construir aplicativos para web baseados em uma única página dinâmica, eu tenho seis meses de exepêriencia com essa tecnologia. E continuo me aprimorando, e estudando essa ferramenta poderosa.
