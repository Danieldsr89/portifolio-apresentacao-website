Conhecimentos

HTML

É código de marcação de texto, que é utilizado na criação de páginas web. Tenho um ano e seis meses de expêriencia com essa tecnologia.

CSS

É uma liguagem de marcação, utilizada na formatação de textos, no layout das páginas web, e no desenvonvimento das cores. CSS, cuida do estilo das páginas web. Tenho um ano e seis meses de exepêriencia com essa tecnologia.

Design

Figma é uma plataforma utilizada na construção do design de interfaces, das páginas web. Tenho seis meses de exepêriencia com essa tecnologia.

Javascript

É uma liguagem de programação utilizada no desenvonvimento de páginas Web. Javascript é mais aplicada na parte visual dos sites, essa linguagem é responsável pelas animações das páginas web. Tenho um ano e seis meses de exepêriencia com essa tecnologia.

GitHub

É uma plataforma de hospedagem de projetos, códigos-fonte e arquivos. GitHub é a rede social, de hospedagem de projetos. Tenho seis meses de exepêriencia com essa tecnologia.
